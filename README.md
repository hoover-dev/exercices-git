# Annuaire Utilisateurs

Application web qui récupère et affiche une liste d'utilisateurs depuis une API.

## Fonctionnalités
- Affichage de 10 utilisateurs (nom, email, ville, téléphone)
- Recherche en temps réel par nom
- Message si aucun résultat trouvé
- Compteur de résultats

## Technologies utilisées
- HTML / CSS
- JavaScript (Fetch API, DOM, boucles, conditions)
- GitHub Pages (déploiement)
